# Mouse Tracking Visuals

This directory stores generated mouse tracking figures and a standalone generator for regenerating them.

Run:

```bash
python mouse_tracking_visuals/build_mouse_tracking_visuals.py --mouse AL032
```

The generator is self-contained in:

- `mouse_tracking_visuals/build_mouse_tracking_visuals.py`

Generated outputs are written to:

- `mouse_tracking_visuals/<mouse>/`
